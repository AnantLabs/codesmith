Products  (Created 8/6/2000 1:34:07 AM)
Description:
This table contains all of the products.
Key ID Name Data Type Size(Precision,Scale) Nulls Default
ProductID int 0
  The primary key for each product.
ProductName nvarchar 40
  The product's name.
SupplierID int 0
  The product's supplier.
CategoryID int 0
  The product's category.
QuantityPerUnit nvarchar 20
  The quantity for each unit of this product.
UnitPrice money 0 (0)
  The price per unit of the product.
UnitsInStock smallint 0 (0)
  The number of units currently in stock for this product.
UnitsOnOrder smallint 0 (0)
  The number of units currently on order for this product.
ReorderLevel smallint 0 (0)
  The level at which an automatic re-order of this product should take place.
Discontinued bit 0 (0)
  A flag indicating if this product has been discontinued.
